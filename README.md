# DNS Router Checker


__DNS Router Checker__ is a simple script that check a provided DNS server the your lan Router. This is just to prevent (warning) DNS home pishing.

### Installing
`git clone https://github.com/moraisaugusto/dnsrc`
`make`
`sudo make install`
